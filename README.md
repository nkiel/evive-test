## Dependencies

- Java >=1.8

## Build

This project uses gradle to build. Included is the gradle wrapper`(gradlew)` which will execute the build tasks locally with Java.

Execute tests:
```
gradlew test
```

Run Application:
```
gradlew run
```
> Application will continually scan STDIN for new orders
